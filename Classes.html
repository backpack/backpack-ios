  • BPKHorizontalNavigationObjc

    A limited version of BPKHorizontalNavigation that can be used from Objective-C.

    BPKHorizontalNavigation isn’t usable from Swift due to its API. This class provides an Objective-C compatible wrapper with limited functionality to use in its place in Objective-C code.

    It’s recommended that the Swift version is used, but this class exists as an alternative.
