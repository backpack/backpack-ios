BPKCalendarSelectionConfigurationRange

@interface BPKCalendarSelectionConfigurationRange
    : BPKCalendarSelectionConfiguration

BPKCalendarSelectionConfigurationRange is a subclass of BPKCalendarSelectionConfiguration designed to enable a user to select a range of dates, represented by a start date and end date.

  • startSelectionHint

    The hint provided to assistive technologies informing a user how to select the first date in the range. This hint will be provided on any cell that can be selected as the first date.
